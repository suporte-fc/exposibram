Hotmart is ideal to who have, or want to create digital paid content online. In a simple, efficient way.

== Description ==

With the Hotlink Manager plugin, you can sell through Hotmart's digital content plataform but with your url's domain. This plugins converts the hotlinks from Hotmart to a link in your domain. 

== Installation ==

1. Upload the 'hotlinks-manager' folder to the '/wp-content/plugins/' directory.
1. Activate the Hotlinks Manager plugin through the 'Plugins' menu in WordPress.
1. Use the shortcode [hotlinks-generator] in a page.
