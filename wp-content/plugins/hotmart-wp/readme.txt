Hotmart's official plugin

== Description ==

With the Hotmart's official plugin you have all the tools that you need to integrate your site with our platform.

== Installation ==

1. Upload the "hotmart-wp.zip" OR copy the "hotmart-wp" folder to "wp-content/plugins";
2. Activate the plugin through the "Plugins" menu;
3. Set your preferences through the "Hotmart WP" menu;
4. All done! Have fun!
